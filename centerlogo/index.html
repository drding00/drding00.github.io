Centered Logos and Such

I've had troubles with coding a responsive menu with a logo in the center of it. I've seen it done here and there, but was never able to achieve it properly myself.

The tricky part is how the menu collapses as the display gets smaller and smaller. Then there's the issue of where the logo goes when you get to just before a mobile view.

This example uses a combo of Flexbox and some absolute positioning. I've left the number of links uneven because that seems like a more realistic scenario.

At 1082px

The logo gets an absolute position and moved up, pulling it from the flex container and allowing the left and right links to settle side by side one another.

At 950px

The desktop navigation is hidden (via display: none;) and a mobile menu is displayed instead.
